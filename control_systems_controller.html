Design and Testing of a Discrete Feedback Control System

This practical involved the design and testing of a discrete feedback control system for a simulated plant. Two different controllers were implemented: a discrete state-space controller with reference input and a discrete state-space controller with integral control. The objective was to evaluate and compare their performance in tracking a reference signal. Additionally, an external disturbance was introduced to assess the disturbance rejection capability of each controller. The analysis focused on key performance metrics, including transient response, steady-state tracking error, and overall system stability. The results demonstrated how effectively each controller followed the reference signal and responded to disturbances.
