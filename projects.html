  • Design and Testing of a Discrete Feedback Control System

    This project involved designing and testing a discrete feedback control system for a simulated plant. Two controllers were implemented: a discrete state-space controller with reference input and a discrete state-space controller with integral control. Their performance was evaluated based on reference tracking, disturbance rejection, and steady-state error. The results highlight how effectively each controller follows the reference signal and handles external disturbances.
